# Summary: OpenAI GPT-5.6 Model Family Launch

**Status:** General Availability (Global rollout starting immediately, completing within 24 hours).
**Source:** OpenAI Official Announcement.

## 1. Product Overview
OpenAI has launched the **GPT-5.6** family, consisting of three durable capability tiers that advance independently of the generation number:
*   **GPT-5.6 Sol:** Flagship model. State-of-the-art intelligence and efficiency. Best for complex coding, science, cybersecurity, and knowledge work.
*   **GPT-5.6 Terra:** Balanced model. Competitive with GPT-5.5 performance at lower cost.
*   **GPT-5.6 Luna:** Cost-efficient model. Fastest and most affordable; outperforms Opus 4.8 on coding benchmarks.

## 2. Key Technical Capabilities & Features
*   **Efficiency:** Trained to maximize utility per token. Sol uses fewer tokens and less time than competitors (e.g., Claude Fable 5) while achieving higher scores on long-horizon professional workflows (Agents’ Last Exam: 53.6 vs. 40.5).
*   **Reasoning Modes:**
    *   `max`: Extended reasoning time for complex problem-solving.
    *   `ultra`: Highest capability setting. Coordinates **four agents in parallel** by default (configurable up to 16) to finish demanding tasks faster. Available in API via Multi-Agent Beta and in ChatGPT Work/Codex for specific tiers.
*   **Programmatic Tool Calling:** New feature in the Responses API allowing GPT-5.6 to write and run lightweight in-memory programs to coordinate tools, filter intermediate data, and adapt workflows. This enables **Zero Data Retention (ZDR)** compatibility.
*   **Computer Use & Design:** Enhanced ability to inspect, refine, and deliver ready-to-use visual results. Can infer design systems (typography, spacing, layouts) from reference decks to create polished presentations and documents.
*   **Cybersecurity:** Significant gains in defensive tasks (secure code review, patching, threat modeling). Scores 73.5% on ExploitBench2 (vs. 47.9% for GPT-5.5). Does **not** cross the "Critical" threshold for autonomous end-to-end attacks against hardened targets.
*   **Science & Biology:** Strong performance in life sciences (GeneBench Pro, LifeSciBench). Supports legitimate research but does not provide end-to-end capability to create/synthesize novel dangerous threats.

## 3. Pricing & API Details
*   **Sol:** $5 input / $30 output per 1M tokens.
*   **Terra:** $2.50 input / $15 output per 1M tokens.
*   **Luna:** $1 input / $6 output per 1M tokens.
*   **Prompt Caching:** More predictable caching with explicit breakpoints and a 30-minute minimum cache life. Cache writes billed at 1.25x uncached input rate; reads receive 90% discount.

## 4. Safety & Safeguards
*   **Layered Defense:** Combines model-trained protections, real-time checks, continuous monitoring, and account-level enforcement.
*   **Reasoning Monitor:** Uses test-time reasoning to review conversations for potential harm, allowing for rapid updates without retraining classifiers.
*   **Cyber Safeguards:** Blocks roughly **10x more** potentially harmful activity than previous models. Designed to allow defensive work (blue teaming, patching) while restricting offensive exploitation.
*   **Trusted Access:** Advanced defensive capabilities in cybersecurity are reserved for verified users via the "Daybreak’s Trusted Access for Cyber" program.
*   **Evaluation:** Underwent extensive red teaming, including ~700,000 A100e GPU hours of automated black-box testing.

## 5. Availability & Access Tiers
*   **ChatGPT:**
    *   **Plus/Pro/Business/Enterprise:** Access Sol (medium+ effort). Pro/Enterprise can select "Sol Pro" for highest quality.
    *   **Free/Go:** Access Terra.
*   **ChatGPT Work & Codex:**
    *   **Free/Go:** Terra.
    *   **Plus/Pro/Business/Enterprise:** Choice of Sol, Terra, or Luna.
    *   **Effort Levels:** `max` available to all. `ultra` available to Pro/Enterprise (Work) and Plus+ (Codex).
*   **API:** All three models (Sol, Terra, Luna) available. Multi-agent beta and Programmatic Tool Calling supported in Responses API.

## 6. Notable Benchmark Performance (Selected)
*   **Coding (Artificial Analysis Coding Agent Index):** Sol 80 (vs. Fable 5 77.2); Terra 77.4; Luna 74.6.
*   **Cybersecurity (ExploitBench2):** Sol 73.5% (vs. GPT-5.5 47.9%).
*   **Computer Use (BrowseComp):** Sol 92.2% (Ultra); Sol 90.4% (Standard).
*   **Science (GeneBench Pro):** Sol 28.7% (vs. GPT-5.5 12%).
*   **Long Context (MRCR v2 1M):** Sol 73.8%; Terra 72.5%; Luna 41.3%.

## 7. Operator Considerations & Risks
*   **Cost Efficiency:** Significant cost reductions compared to previous generations, especially for Luna and Terra. Ideal for high-volume, lower-complexity tasks.
*   **Safety Friction:** Cyber safeguards are significantly stricter. Users may encounter more blocks for benign prompts; retry options on lower-capability models are provided.
*   **Implementation:** Leverage **Programmatic Tool Calling** for ZDR compliance and efficient tool orchestration. Use **Multi-Agent** (`ultra`) only for tasks requiring parallel processing to justify higher token costs.
*   **Dual-Use Nature:** While improved for defensive cybersecurity and biology, the models remain dual-use. Ensure appropriate access controls and monitoring for internal deployments, especially in sensitive domains.
