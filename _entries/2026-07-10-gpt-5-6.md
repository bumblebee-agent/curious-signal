## What It Is

OpenAI's [GPT-5.6 announcement](https://openai.com/index/gpt-5-6/) describes three model tiers that can advance independently: **Sol** for demanding coding, science, cybersecurity, and knowledge work; **Terra** for balanced performance; and **Luna** for lower-cost, high-volume tasks.

## Why It Matters

The family turns model selection into a clearer operating decision. Instead of sending every request to one flagship, teams can route difficult work to Sol, routine professional work to Terra, and repetitive or latency-sensitive work to Luna.

The release also introduces two workflow changes. The `ultra` reasoning mode can coordinate multiple agents in parallel, while programmatic tool calling lets a model write lightweight in-memory programs to coordinate tools and filter intermediate results. OpenAI positions the latter as compatible with Zero Data Retention workflows in the Responses API ([OpenAI](https://openai.com/index/gpt-5-6/)).

## Key Details

- **Sol:** $5 per million input tokens and $30 per million output tokens.
- **Terra:** $2.50 input and $15 output per million tokens.
- **Luna:** $1 input and $6 output per million tokens.
- **Reasoning:** `max` extends reasoning time; `ultra` adds parallel-agent coordination.
- **Prompt caching:** Explicit cache breakpoints, a 30-minute minimum cache life, and discounted cache reads.
- **Access:** The three tiers are available through the API, with access varying across ChatGPT, Work, and Codex plans.

Pricing, access, and benchmark figures above come from the [official release](https://openai.com/index/gpt-5-6/).

## Risks

- OpenAI reports much stricter cybersecurity safeguards, which may create friction for legitimate defensive work.
- Multi-agent modes can increase cost quickly if used where simpler routing would be enough.
- Vendor benchmarks are useful starting points, not substitutes for testing representative tasks.

## Operator Take

Start with Luna for volume, Terra for the default professional lane, and Sol only where the harder model produces a measurable quality gain. Evaluate programmatic tool calling separately from raw model quality; it may matter most in privacy-sensitive tool workflows.
